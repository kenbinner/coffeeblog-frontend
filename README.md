# Coffee Blog - Front end

Frontend of my coffee blog website created using React.

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

### `npx json-server --watch testdatabase.json --port 400`

Runs the json server with test data for testing axios functions.\
Open [http://localhost:4000](http://localhost:4000) to view it in the browser.
